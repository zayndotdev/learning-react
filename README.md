## 💡 How to Run Locally

1. Clone the repo:

   ```bash
   git clone https://github.com/zayndotdev/learning-react.git
   cd learning-react
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev   # or `npm start` if using Create React App
   ```
